What is HTML?

⌄

HTML stands for HyperText Markup Language. It's the standard language for creating webpages and web applications.

What is CSS?

⌄

CSS stands for Cascading Style Sheets. It controls the look and feel of a website — layout, colors, fonts, and more.

What is JavaScript?

⌄

JavaScript is a programming language that allows you to add interactivity to your website — like animations, sliders, and logic.
